## Getting Started

- ## Requirements

  - [x] Dotnet v6.0.8

- ## Setup

  - ### Clone

    ```shell
    git clone https://github.com/YourIbexCapra/RubiX.git
    ```

  - ## Start

    ```shell
    dotnet run <args>
    ```

- ## Build

  ```shell
  dotnet build
  ```
